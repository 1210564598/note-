2.数据类型

## **一、常量与变量**

### 1. 核心概念
- 常量：程序运行中值不可改，分为整型（100、-200）、实型（3.14、0.125）、字符型（'a'、'\\n'）、字符串型（"abc"）；宏定义常量（#define MAX 10）、const 常量（const int b=10）均不可修改。
- 变量：程序运行中值可变，需先定义后使用（格式：类型 变量名），编译时分配内存，可通过名字或地址访问。

## 2. 变量命名规则
- 不能是关键字（如 int、char）；
- 由字母、数字、下划线组成，首字符必须是字母或下划线；
- 区分大小写（Age≠age）。

### 3. 声明与定义区别
- 声明：不分配存储空间（如extern int a），仅告知变量存在；
- 定义：分配存储空间（如int b=10），兼具声明作用。

## **二、整型（int）**

### 1. 整型分类与占用空间

| **类型** | **占用空间** | **取值范围** |
| --- | --- | --- |
| short（短整型） | 2 字节 | -32768~32767 |
| int（整型） | 4 字节 | -2147483648~2147483647 |
| long（长整型） | Windows=4 字节；Linux=4/8 字节 | 同 int 或更大范围 |
| long long | 8 字节 | -9223372036854775808~9223372036854775807 |
| unsigned 修饰 | 同对应类型 | 非负数（如 unsigned int：0~4294967295） |

### 2. 赋值与输入输出
- 赋值格式：十进制（123）、八进制（0123）、十六进制（0x123）；
- 输出格式：%d（有符号十进制）、%o（八进制）、%x/%X（十六进制）、%u（无符号十进制）；
- 输入格式：scanf("%d", &变量名)（& 为取地址符，不可省略）。

### 3. 有符号与无符号区别
- 有符号数：最高位为符号位（0 = 正，1 = 负），可存正负值；
- 无符号数：最高位为数值位，仅存非负数，取值范围翻倍。

## **三、sizeof 关键字**
- 功能：计算数据类型 / 变量的占用字节数，非函数，无需头文件；
- 返回值：size_t 类型（32 位为 unsigned int）；
- 示例：sizeof(int)=4、sizeof(char)=1。

## **四、字符型（char）**
- 1. 核心特性
- 占用 1 字节，本质是 1 字节整型，存储字符的 ASCII 编码；
- 赋值需用单引号（char ch='a'），不可用双引号。
- 2. 输入输出与转换
- 输出：%c（打印字符）、%d（打印 ASCII 码）；
- 大小写转换：'a'-32='A'、'A'+32='a'（空格 ASCII 码为 32，也可通过'a'-' '转换）。
- 3. 转义字符（常用）

| -   转义字符 | -   含义 | -   ASCII 码 |
| --- | --- | --- |
| -   \\n | -   换行 | -   010 |
| -   \\t | -   水平制表 | -   009 |
| -   \\r | -   回车 | -   013 |
| -   \\ | -   反斜线 | -   092 |
| -   ' | -   单引号 | -   039 |
| -   " | -   双引号 | -   034 |
| -   \\0 | -   字符串结束符 | -   000 |

## **五、实型（浮点型）**
- 1. 分类与特性

| -   类型 | -   占用空间 | -   有效数字 | -   赋值格式 |
| --- | --- | --- | --- |
| -   float | -   4 字节 | -   6~7 位 | -   需加 f 后缀（3.14f） |
| -   double | -   8 字节 | -   15~16 位 | -   默认浮点型（3.14） |
- 2. 赋值与输出
- 科学计数法：3.2e3f=3200、100e-3f=0.1；
- 输出：%f（float/double）、%lf（double），默认保留 6 位小数。

## **六、进制**
- 1. 进制表示与转换

| -   进制 | -   C 语言表示 | -   转换方法 |
| --- | --- | --- |
| -   十进制 | -   123 | -   直接书写 |
| -   八进制 | -   0123 | -   十进制数 ÷8，余数倒序排列 |
| -   十六进制 | -   0x123 | -   十进制数 ÷16，余数倒序（10~15 用 A~F） |
| -   二进制 | -   无直接表示 | -   十进制数 ÷2，余数倒序排列 |
- 2. 常用单位
- 1bit（位）=1 二进制位；1Byte（字节）=8bit；
- 1KB=1024Byte，1MB=1024KB，1GB=1024MB，1TB=1024GB。

## **七、内存数值存储方式**
- 1. 原码、反码、补码
- 正数：三者相同（符号位 0，数值位为绝对值二进制）；
- 负数：原码（符号位 1，数值位为绝对值二进制）→ 反码（符号位不变，数值位取反）→ 补码（反码 + 1）；
- 计算机统一用补码存储，解决 0 的二义性（+0 和 - 0 补码均为 0000 0000），将减法转为加法。
- 2. 数值溢出
- 有符号数：溢出后正负反转（如 char 型 127+2=-127）；
- 无符号数：溢出后高位丢失（如 unsigned char 型 255+1=0）。

## **八、类型限定符**

| -   限定符 | -   含义 |
| --- | --- |
| -   extern | -   声明变量，不分配存储空间 |
| -   const | -   定义常量，值不可修改 |
| -   volatile | -   防止编译器优化代码（如硬件寄存器变量） |
| -   register | -   建议将变量存入 CPU 寄存器，提高效率 |

## **九、字符串格式化输入输出**
- 1. 字符串常量
- 以'\\0'为结束符（编译器自动添加），如 "a" 占用 2 字节（'a'+'\\0'）。
- 2. 输出函数
- printf：支持多格式输出，常用格式符%d（int）、%c（char）、%f（浮点型）、%s（字符串）、%p（指针地址）；
- putchar：仅输出单个字符（putchar('a')）。
- 3. 输入函数
- scanf：输入时格式符需与变量类型匹配，如scanf("%c", &ch)、scanf("%d", &a)；
- getchar：读取单个字符，可用于吸收输入缓冲区的回车符。
