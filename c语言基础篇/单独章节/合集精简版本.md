# C 语言基础篇合集精简版本

## 第一章 C 语言概述

1. **核心特性**：代码量小、执行快、功能强；开发周期长、可移植性差，32 个关键字、9 种控制语句、34 种运算符。
2. **应用领域**：操作系统 / 驱动、嵌入式、高性能服务、数据库、编译器等。
3. **第一个程序**：源文件后缀`.c`，`main()`为唯一入口，`printf`依赖`<stdio.h>`，`return 0`表示执行成功。
4. **GCC 编译**：核心命令`gcc 源文件.c -o 可执行文件`，四步流程：预处理（-E）→编译（-S）→汇编（-c）→链接。
5. **IDE 快捷键**：Qt Creator（Ctrl+i 格式化、Ctrl+/ 注释）；VS（Ctrl+k+f 格式化、Ctrl+F5 不调试运行）。
6. **system 函数**：头文件`<stdlib.h>`，Windows`system("calc")`、Linux`system("ls")`。

## 第二章 数据类型

1. **常量与变量**：常量值不可改（`#define`/`const`）；变量先定义后使用，命名含字母 / 数字 / 下划线，首字符非数字。
2. 基本类型
   - 整型：short (2B)、int (4B)、long long (8B)，`unsigned`修饰为非负数；
   - 字符型：char (1B)，存 ASCII 码，转义字符`\n`/`\t`/`\0`（字符串结束符）；
   - 浮点型：float (4B，6~7 位有效数字)、double (8B，15~16 位有效数字)。
3. **sizeof**：计算字节数，非函数，无需头文件。
4. **存储方式**：计算机用补码存储，正数原 / 反 / 补码相同，负数补码 = 反码 + 1。
5. **类型限定符**：`extern`（声明）、`const`（常量）、`register`（寄存器变量）、`volatile`（防止优化）。
6. **输入输出**：`printf/scanf`多格式输出输入，`putchar/getchar`单字符操作。

## 第三章 运算符与表达式

1. **六大运算符**：算术（++/--/%）、赋值（+=/-= 等）、比较（==/!=/>=）、逻辑（&&/||/!）、位运算、sizeof。
2. **自增自减**：前置（先变后用）、后置（先用后变）；% 仅适用于整型。
3. **优先级**：括号 > 单目运算符 > 算术 > 比较 > 逻辑 > 三目 > 赋值 > 逗号（数字越小优先级越高）。
4. **类型转换**：隐式（低精度→高精度，自动）；强制（`(类型)表达式`，临时转换，不改变原变量）。
5. **逻辑判断**：0 为假，非 0 为真；&& 短路与（前假后不执行）、|| 短路或（前真后不执行）。

## 第四章 程序流程结构 + 数组 + 字符串

流程结构

1. **三大结构**：顺序（依次执行）、选择（if / 三目 /switch）、循环（while/do-while/for）。
2. **选择**：switch 匹配整型 / 字符型，`case`后需`break`防穿透；三目运算符`条件?表达式1:表达式2`。
3. **循环**：while 先判断后执行，do-while 至少执行一次，for 适合循环次数明确的场景。
4. **跳转语句**：`break`跳出当前循环 /switch；`continue`跳过本次循环；`goto`尽量少用。

数组

1. **核心特性**：相同类型、内存连续、下标从 0 开始，数组名为地址常量，不可赋值。
2. **一维数组**：定义`类型 数组名[常量]`，初始化局部数组默认随机值，全局数组默认 0，`int a[]={1,2}`自动推导长度。
3. **二维数组**：定义`类型 数组名[行][列]`，按行存储，行数 =`sizeof(a)/sizeof(a[0])`，列数 =`sizeof(a[0])/sizeof(a[0][0])`。
4. **常用操作**：求最值、数组逆置、冒泡排序（相邻比较，逐步冒泡）。

字符串

1. **本质**：以`\0`结尾的字符数组，无专门字符串类型，`\0`不占有效长度。
2. **初始化**：推荐`char c[]="abc"`（自动加`\0`），`char c[]={'a','b','c'}`无`\0`，输出乱码。
3. **输入输出**：推荐`fgets/fputs`（安全，防溢出），`strlen`计算有效长度（不含`\0`），头文件`<string.h>`。

## 第五章 函数

1. **核心作用**：代码复用、模块化，分库函数和自定义函数。
2. **定义格式**：`返回类型 函数名(形参列表){函数体}`，无返回值用`void`，`return`可中断函数。
3. **调用规则**：实参与形参类型 / 个数一致，值传递形参修改不影响实参；声明格式`返回类型 函数名(形参类型);`（分号必加）。
4. **main 与 exit**：main 中`return 0`和`exit(0)`均终止程序；子函数中`return`仅终止自身，`exit`终止整个程序（头文件`<stdlib.h>`）。
5. **多文件编程**：.h 存声明，.c 存实现；防头文件重复包含：`#ifndef...#define...#endif`或`#pragma once`。

## 第六章 指针

1. **核心概念**：指针 = 内存地址，指针变量存地址，类型决定步长（int*+1 移动 4B），32 位平台指针占 4B，64 位占 8B。
2. **基础操作**：`&`取地址，`*`解引用；野指针（未初始化 / 指向已释放内存）需避免，空指针`NULL`（值为 0）。
3. **const 修饰指针**：`const int*p`（不可改指向的值）、`int*const p`（不可改指向）、`const int*const p`（均不可改）。
4. **指针与数组**：数组名 = 首元素地址常量，`a[i]`等价于`*(a+i)`；指针数组`int*p[10]`（数组元素为指针）。
5. **指针与函数**：地址传递（传指针）可修改实参；数组名传参退化为指针，需手动传长度；返回值避免栈区地址。
6. **指针与字符串**：`char*p="abc"`指向只读常量，不可修改；`char p[]="abc"`为字符数组，可修改；常用函数`strcpy/strcat/strcmp/strstr`。
7. **多级指针**：二级指针`int**p`，指向一级指针的地址，核心`**q=*p=a`。

## 第七章 内存管理

1. 作用域与生命周期
   - 局部变量（auto）：代码块内，函数执行后销毁，栈区；
   - 静态局部变量：代码块内，程序运行期有效，data/bss 区；
   - 全局变量：整个程序，data/bss 区；静态全局变量：仅当前文件有效；
   - static 函数：仅当前文件可调用，不同文件可重名。
2. **内存分区**：加载前（代码区、data 区、bss 区）；加载后新增栈区（自动分配）、堆区（手动分配）。
3. **内存操作函数**：头文件`<string.h>`，`memset`（填充）、`memcpy`（拷贝，不重叠）、`memmove`（拷贝，支持重叠）、`memcmp`（比较）。
4. **堆区操作**：`malloc`（分配内存，返回首地址，需强制转换）、`free`（释放内存，后置 NULL 防野指针），头文件`<stdlib.h>`。
5. **常见错误**：返回栈区地址（野指针）、值传递修改指针本身、多次释放堆内存。

## 第八章 复合类型

1. 结构体
   - 定义：组合不同类型数据，类型不占内存，变量占内存，三种定义方式（先声明后定义 / 声明同时定义 / 无类型名定义）。
   - 访问：普通变量用`.`，指针变量用`->`；结构体数组`struct stu s[5]`，通过下标 +`.`访问。
   - 赋值：同类型结构体可直接赋值；堆区结构体`struct stu*p=(struct stu*)malloc(sizeof(struct stu))`，使用后需 free。
   - 传参：地址传递（传指针）效率高，`const`修饰形参保护实参。
2. **共用体**：所有成员共享同一块内存，大小 = 最长成员大小，同一时刻仅一个成员有效，地址与所有成员地址相同。
3. **枚举**：列举变量所有可能值，默认从 0 开始，可手动指定，枚举值为常量，不可修改。
4. **typedef**：为已有类型起别名，不创建新类型，编译阶段生效，与`#define`（预处理，字符串替换）不同。

## 第九章 文件操作

1. **文件分类**：磁盘文件（文本 / 二进制）、设备文件（键盘 / 屏幕）；文本文件存 ASCII 码，二进制文件存原始值，更高效。
2. **文件指针**：`FILE*`，系统默认`stdin`（键盘）、`stdout`（屏幕）、`stderr`（屏幕）。
3. **打开与关闭**：`fopen(文件名, 模式)`，模式`r`（只读）、`w`（只写，清空 / 创建）、`a`（追加），加`b`为二进制模式；`fclose`关闭文件，必做，防内存泄漏 / 数据丢失。
4. 顺序读写
   - 按字符：`fputc`（写）、`fgetc`（读）；
   - 按行：`fputs`（写）、`fgets`（读，安全）；
   - 格式化：`fprintf`（写）、`fscanf`（读）；
   - 按块：`fwrite/fread`（二进制文件核心，效率最高），头文件`<stdio.h>`。
5. **随机读写**：`fseek`（移动光标，`SEEK_SET`/`SEEK_CUR`/`SEEK_END`）、`ftell`（获取光标位置，计算文件大小）、`rewind`（光标移开头）。
6. **其他操作**：`remove`（删除文件）、`rename`（重命名 / 移动）、`fflush`（刷新缓冲区，立即写入磁盘）。
7. **缓冲区**：缓冲文件系统，程序先操作缓冲区，再刷入磁盘，减少 I/O 次数。

## 第十章 补充核心要点

1. **关键头文件**：`<stdio.h>`（输入输出）、`<stdlib.h>`（system/malloc/exit）、`<string.h>`（字符串 / 内存操作）、`<time.h>`（随机数种子）。
2. **常见错误**：C4996（VS，加`#define _CRT_SECURE_NO_WARNINGS`）、野指针、缓冲区溢出、堆内存未释放、数组下标越界。
3. **开发规范**：变量见名知意，函数单一职责，多文件编程分离声明与实现，及时释放堆内存，避免 goto 滥用。
