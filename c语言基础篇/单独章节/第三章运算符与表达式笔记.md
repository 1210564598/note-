# 第三章 运算符与表达式

## 一、常用运算符分类

C 语言运算符按功能可分为 6 大类，核心作用各有侧重，是构成表达式的基础：

| **运算符类型** | **核心作用** |
| --- | --- |
| 算术运算符 | 处理四则运算、自增自减、取模等数值计算 |
| 赋值运算符 | 将表达式计算结果赋给变量，含复合赋值 |
| 比较运算符 | 比较表达式大小 / 相等关系，返回真 (1)/ 假 (0) |
| 逻辑运算符 | 对布尔值（0/1）进行逻辑判断，返回真 (1)/ 假 (0) |
| 位运算符 | 对数据的二进制位进行移位、与 / 或 / 异或等操作 |
| sizeof 运算符 | 计算数据 / 变量 / 类型占用的内存字节数 |

## 二、核心运算符详解

### （一）算术运算符

包含基础四则、取模、自增自减，****自增自减分前后，结果差异关键****，取模仅适用于整型：

| **运算符** | **术语** | **示例** | **结果** |
| --- | --- | --- | --- |
| + | 正号 | +3 | 3 |
| - | 负号 | -3 | -3 |
| + | 加 | 10+5 | 15 |
| - | 减 | 10-5 | 5 |
| * | 乘 | 10*5 | 50 |
| / | 除 | 10/5 | 2 |
| % | 取模 | 10%3 | 1 |
| ++ | 前自增 | a=2;b=++a; | a=3;b=3 |
| ++ | 后自增 | a=2;b=a++; | a=3;b=2 |
| -- | 前自减 | a=2;b=--a; | a=1;b=1 |
| -- | 后自减 | a=2;b=a--; | a=1;b=2 |

### （二）赋值运算符

基础赋值为=，复合赋值是基础运算+=/-=/*=/=/=%=，简化运算赋值一步操作：

| **运算符** | **术语** | **示例** | **结果** |
| --- | --- | --- | --- |
| = | 赋值 | a=2;b=3; | a=2;b=3 |
| += | 加等于 | a=0;a+=2; | a=2 |
| -= | 减等于 | a=5;a-=3; | a=2 |
| *= | 乘等于 | a=2;a*=2; | a=4 |
| /= | 除等于 | a=4;a/=2; | a=2 |
| %= | 模等于 | a=3;a%=2; | a=1 |

### （三）比较运算符

****C 语言中真用 1 表示，假用 0 表示****，仅判断关系，不做数值计算：

| **运算符** | **术语** | **示例** | **结果** |
| --- | --- | --- | --- |
| == | 相等于 | 4==3 | 0 |
| != | 不等于 | 4!=3 | 1 |
| < | 小于 | 4<3 | 0 |
| \> | 大于 | 4>3 | 1 |
| <= | 小于等于 | 4<=3 | 0 |
| \>= | 大于等于 | 4>=1 | 1 |

### （四）逻辑运算符

操作数为 0（假）或非 0（真），核心是逻辑与 / 或 / 非的判断规则：

| 运算符 | 术语 | 示例 | 结果规则 |
| --- | --- | --- | --- |
| ! | 非 | !a | a 假则！a 真，a 真则！a 假 |
| && | 与 | a&&b | 两者都真则真，否则假 |
| \|\ | |或 | a\|\|b | 其一为真则真，两者都假则假 |

## 三、运算符优先级与结合方向

优先级决定运算执行顺序（****数字越小，优先级越高****），结合方向决定同优先级运算的执行顺序，核心规则如下：

. 最高优先级（1）：数组下标[]、圆括号()、成员选择. / ->，左到右结合
. 单目运算符（2）：负号-、按位取反~、自增++/ 自减--、取值*、取地址&、逻辑非!、强制类型转换(类型)、sizeof，右到左结合
. 算术乘除模（3）：/ \* %，左到右结合（双目）
. 算术加减（4）：+ -，左到右结合（双目）
. 位移位（5）：<< >>，左到右结合（双目）
. 比较大小（6）：> >= < <=，左到右结合（双目）
. 比较相等（7）：== !=，左到右结合（双目）
. 按位与（8）：&，左到右结合（双目）
. 按位异或（9）：^，左到右结合（双目）
. 按位或（10）：|，左到右结合（双目）
. 逻辑与（11）：&&，左到右结合（双目）
. 逻辑或（12）：||，左到右结合（双目）
. 三目运算符（13）：条件运算符?:，右到左结合（唯一三目）
. 赋值运算符（14）：基础赋值=、复合赋值+=/-=/*=/等，右到左结合
. 最低优先级（15）：逗号运算符,，左到右结合

## 四、类型转换

不同类型数据混合运算时需进行类型转换，遵循****精度不降低****原则（值域小→值域大，字节数少→字节数多），分两种方式：

### （一）隐式转换（自动转换）

由编译系统自动完成，无需手动干预，运算数类型不同时，低精度向高精度转换。

示例代码：

```c

#include <stdio.h>
int main()
{
    int num = 5;
    printf("s1=%d\n", num / 2);    // 整型相除，结果为整型2
    printf("s2=%lf\n", num / 2.0); // 整型与浮点型运算，num自动转为浮点型，结果2.500000
    return 0;
}

```



### （二）强制类型转换

手动使用强制类型转换运算符完成，强制将表达式结果转为指定类型，语法：(类型说明符) (表达式)。注意：仅临时转换运算结果，不改变原变量的类型和值。

示例代码：

```c
#include <stdio.h>
int main()
{
    float x = 0;
    int i = 0;
    x = 3.6f;
    i = (int)x;  // 强制将浮点型3.6转为整型3，i=3
    printf("x=%f, i=%d\n", x, i); // 输出x=3.600000, i=3
    return 0;
}
```
