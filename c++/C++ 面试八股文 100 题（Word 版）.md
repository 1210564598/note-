# C++ 面试八股文 100 题（Word 版）

## 一、C++ 基础与语法（20 题）

### 1. C++11 新特性有哪些？

问答：auto/decltype、右值引用、移动语义、lambda、智能指针、nullptr、范围 for、constexpr、列表初始化。

速记：自动推导、移动语义、智能指针、lambda。

### 2. auto 作用是什么？

问答：编译期自动推导变量类型，简化代码，必须初始化。

速记：自动推导类型。

### 3. 指针和引用区别？

问答：指针可空、可改指向、多级；引用必须初始化、不能为空、不可重绑定。

速记：指针可变，引用不可变。

### 4. const 与 #define 区别？

问答：const 有类型检查、有作用域、可调试；#define 纯文本替换，无检查。

速记：const 安全，define 危险。

### 5. inline 作用？

问答：编译期展开函数体，减少调用开销，比宏更安全。

速记：展开代码，省开销。

### 6. 函数重载条件？

问答：同一作用域、同名、参数不同（个数 / 类型 / 顺序）。

速记：同名不同参。

### 7. 缺省参数规则？

问答：必须从右向左定义默认值，调用从左向右传参。

速记：右→左定义。

### 8. static 作用？

问答：局部变量延长生命周期；全局变量限制文件域；成员无 this 指针。

速记：延长生命周期、限制作用域。

### 9. const 成员函数意义？

问答：不修改成员变量，const 对象可调用。

速记：只读函数。

### 10. typedef vs using？

问答：using 更现代，支持模板别名；typedef 不支持。

速记：using 更强。

### 11. C++ 内存分布？

问答：栈、堆、全局 / 静态区、常量区、代码区。

速记：栈堆全局常量代码。

### 12. new/delete 与 malloc/free 区别？

问答：new 调用构造，malloc 不调用；new 自动算大小，返回类型安全。

速记：new 会构造，malloc 不会。

### 13. 野指针是什么？

问答：指向非法 / 已释放内存的指针；避免方式：初始化、delete 后置空。

速记：非法指针，及时置空。

### 14. nullptr 与 NULL 区别？

问答：NULL 是 0，nullptr 是指针类型，更安全。

速记：nullptr 类型安全。

### 15. C++ 四种类型转换？

问答：static_cast、dynamic_cast、const_cast、reinterpret_cast。

速记：静动裁重。

### 16. explicit 作用？

问答：禁止单参构造隐式转换。

速记：禁止隐式构造。

### 17. 右值引用作用？

问答：绑定临时对象，实现移动语义，减少拷贝。

速记：转移资源，不拷贝。

### 18. lambda 语法？

问答：



捕获

File

-> 返回值 {函数体}。

速记：捕获参数返回值体。

### 19. C++ 编译流程？

问答：预处理→编译→汇编→链接。

速记：预编汇编链。

### 20. RAII 是什么？

问答：利用对象生命周期管理资源，构造申请、析构释放。

速记：资源托管给对象。

## 二、面向对象（20 题）

### 21. 面向对象三大特性？

问答：封装、继承、多态。

速记：封继多。

### 22. 类的 6 个默认成员函数？

问答：构造、析构、拷贝构造、赋值重载、取地址、const 取地址。

速记：构析拷赋取址。

### 23. 构造函数能是虚函数吗？

问答：不能，虚表指针在构造后才生成。

速记：构造时无虚表。

### 24. 析构函数为什么要虚？

问答：基类指针删派生类对象，防止内存泄漏。

速记：防泄漏，保证全析构。

### 25. 浅拷贝 vs 深拷贝？

问答：浅拷贝共用内存；深拷贝独立开空间。

速记：浅共用，深独立。

### 26. 拷贝构造 vs 赋值重载？

问答：拷贝构造创建新对象；赋值重载给已有对象赋值。

速记：新建拷，旧赋。

### 27. 赋值重载步骤？

问答：判断自赋值→释放旧空间→开新空间→拷贝→返回 * this。

速记：判自释拷返。

### 28. 三种继承方式？

问答：public、protected、private。

速记：公护私。

### 29. 菱形继承问题？

问答：数据冗余、二义性；解决：虚继承。

速记：虚继承解决。

### 30. 构造析构顺序？

问答：构造：基类→成员→派生；析构相反。

速记：基成派，析反。

### 31. 多态原理？

问答：虚函数 + 虚表 + 虚指针，运行时动态绑定。

速记：虚表动态绑定。

### 32. 多态三条件？

问答：基类虚函数、派生类重写、基类指针 / 引用指向派生类。

速记：虚重指。

### 33. 重载 / 重写 / 重定义？

问答：重载同域不同参；重写跨域同参 virtual；重定义隐藏。

速记：重载重写重定义。

### 34. 纯虚函数与抽象类？

问答：virtual void f ()=0; 抽象类不能实例化。

速记：=0 抽象类。

### 35. 友元作用？

问答：外部可访问私有成员，破坏封装。

速记：破封装，方便访问。

### 36. 单例模式两种实现？

问答：饿汉式（启动创建）、懒汉式（首次使用创建）。

速记：饿汉急，懒汉慢。

### 37. 委托构造？

问答：构造函数调用同类构造。

速记：构造调构造。

### 38. 虚基类作用？

问答：解决菱形继承数据冗余。

速记：虚基类一份数据。

### 39. this 指针？

问答：指向当前对象，非静态成员才有。

速记：指向自己。

### 40. 拷贝构造参数必须是引用？

问答：传值会无限递归调用。

速记：必须引用，防递归。

## 三、内存管理与智能指针（20 题）

### 41. 内存泄漏？

问答：动态内存未释放；检测：valgrind、ASan。

速记：忘释放，工具查。

### 42. 智能指针作用？

问答：自动释放堆内存，避免泄漏。

速记：自动管内存。

### 43. unique_ptr？

问答：独占指针，不能拷贝，可移动。

速记：独占，不可拷贝。

### 44. shared_ptr 原理？

问答：引用计数，共享所有权。

速记：引用计数共享。

### 45. shared_ptr 线程安全？

问答：计数安全，对象不安全。

速记：计数安全，数据不安全。

### 46. weak_ptr 作用？

问答：解决 shared_ptr 循环引用。

速记：破环引用。

### 47. 循环引用？

问答：互相持有，计数无法归零；解决：weak_ptr。

速记：互相持有，用 weak 破。

### 48. make_shared 优点？

问答：一次分配，异常安全。

速记：一次分配更安全。

### 49. 悬空指针？

问答：指向已释放内存。

速记：指向死人。

### 50. 栈 vs 堆？

问答：栈自动、快、小；堆手动、慢、大。

速记：栈快堆大。

### 51. 内存碎片？

问答：大量小空闲内存；解决：内存池。

速记：碎片用池解决。

### 52. 栈溢出？

问答：递归太深、栈空间不足。

速记：递归深，爆栈。

### 53. auto_ptr 问题？

问答：拷贝转移所有权，已废弃。

速记：行为诡异，废弃。

### 54. 裸指针构造多个 shared_ptr？

问答：会重复释放，崩溃。

速记：禁止共用裸指针。

### 55. shared_ptr 删除器？

问答：自定义释放逻辑，如文件、套接字。

速记：自定义释放。

### 56. 什么是 allocator（分配器）？

问答：STL 容器的内存分配工具，负责内存的申请、释放和对象的构造、析构，将内存管理与容器逻辑分离。

速记：STL 专用内存管理工具。

### 57. C++17 std::optional 作用？

问答：表示一个值可能存在或不存在，避免使用 NULL 或特殊值表示 “无意义”，类型安全。

速记：安全表示值存在 / 不存在。

### 58. 定位 new？

问答：在已有内存上构造对象。

速记：原地构造。

### 59. 内存屏障？

问答：防止 CPU 乱序执行，保证多线程可见性。

速记：禁止重排。

### 60. 零拷贝？

问答：减少用户态内核态拷贝，提升效率。

速记：少拷贝，快 IO。

## 四、STL 容器与算法（20 题）

### 61. vector 底层？

问答：动态连续数组。

速记：连续数组。

### 62. vector 扩容？

问答：1.5 倍 / 2 倍扩容，开新空间→拷贝→释放旧。

速记：倍数扩容。

### 63. vector 迭代器失效？

问答：插入、删除、扩容导致。

速记：结构变，迭代器废。

### 64. list 底层？

问答：双向链表。

速记：双向链表。

### 65. deque 底层？

问答：分段连续数组。

速记：分段数组。

### 66. map/set 底层？

问答：红黑树。

速记：红黑树。

### 67. unordered_map 底层？

问答：哈希表。

速记：哈希表。

### 68. 红黑树特性？

问答：黑根、黑叶子、红节点孩子黑、黑高相同。

速记：黑红规则。

### 69. 哈希冲突解决？

问答：链地址、开放寻址、再哈希。

速记：链式 / 开放 / 再哈希。

### 70. string 底层？

问答：char 数组，SSO 短字符串优化。

速记：字符数组 + SSO。

### 71. 迭代器种类？

问答：输入、输出、前向、双向、随机访问。

速记：五迭代器。

### 72. queue/stack？

问答：容器适配器，底层默认 deque。

速记：适配器。

### 73. priority_queue？

问答：大顶堆，底层 vector。

速记：堆结构。

### 74. sort 实现？

问答：快排 + 插排 + 堆排混合。

速记：混合排序。

### 75. 迭代器失效？

问答：容器结构改变，迭代器非法。

速记：结构变，失效。

### 76. resize vs reserve？

问答：resize 开空间 + 初始化；reserve 只开容量。

速记：resize 初始化，reserve 不。

### 77. 空间配置器 allocator？

问答：STL 内存分配器，分离内存与对象。

速记：STL 内存管家。

### 78. set 迭代器 const？

问答：防止修改破坏树结构。

速记：不能改值。

### 79. emplace_back vs push_back？

问答：原地构造，减少拷贝。

速记：原地构造更快。

### 80. STL 算法特点？

问答：通过迭代器操作，不修改容器大小。

速记：通用迭代器算法。
